# mini-apps-mall
微信小程序 商城

## 前言
无意中在慕课网看到一个小程序商城的视频教程，居然要收388元，真是太贵了。看到上面已经贴了个小程序二维码demo，既然有了，那就来照着做一个练练手吧。

## 功能
- [x] 首页
- [x] 分类
- [x] 购物车
- [x] 个人中心
- [x] 商品列表
- [x] 商品详情
- [x] 订单
- [x] 地址管理
